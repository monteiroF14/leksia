# Requisitos do Projeto

## Requisitos Funcionais

| ID | Requisito |
|----|-----------|
| RF-01 | O utilizador deve poder fazer login via GitHub OAuth |
| RF-02 | O utilizador deve poder carregar um arquivo PDF |
| RF-03 | O utilizador deve poder carregar uma URL da Wikipedia |
| RF-04 | O utilizador deve poder interagir com a app via API de Text-to-Speech |
| RF-05 | A app deve possuir sistema de gamificação |

## Requisitos Não-Funcionais

| ID | Requisito |
|----|-----------|
| RNF-01 | O projeto deve ser desenvolvido com JS e CSS/TailwindCSS/Bootstrap |
| RNF-02 | O projeto deve utilizar JSON Server para persistência de dados |
